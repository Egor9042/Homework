# Подсказка по Git

Создание репозитория:
```sh
git init
```

Сохранение для создания коммита:
```sh
git add
```

Создание коммита с комментарием:
```sh
git commit -m "Message"
```

Перечень коммитов:
```sh
git log
```

Сокращённый перечень коммитов:
```sh
git log --oneline
```

Переключение на определённый коммит:

```sh
git checkout название коммита
```
